Conseguir el Nombre del Equipo y el Nombre Usuario en Windows
===========

* Nombre Equipo

```
Inicio > Equipo > Click derecho Propiedades 
```

* Nombre Usuario

```
Panel de control > Cuentas de usuario > Cuentas de usuario 
```

o

```
Inicio + R > cmd > echo %username%
```

* Otra forma

```
Inicio + R > cmd > whoami
```
